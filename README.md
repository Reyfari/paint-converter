# Paint Converter

A free tool for finding equivalent paints across miniature painting brands (Army Painter, Citadel, Two Thin Coats, Vallejo).

Search for a paint, and the converter shows you cross-brand equivalents sourced from vendor-published conversion charts. Matches are classified as **direct** (single chart entry) or **derived** (inferred by chaining two entries, with more uncertainty). Results can be exported to PDF.

Runs entirely in the browser — no backend, no accounts, no tracking.

**From [Project Librarium](https://github.com/Reyfari)** — an AI-powered workshop for managing miniature painting projects (in development).
